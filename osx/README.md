# OSX

## homebrew management

### setup

```bash
osx/homebrew/bootstrap.sh
```

### install Formulas

```bash
osx/homebrew/install.sh common emacs
```

## zsh

```bash
ln -s ~/dotfiles/osx/.zshrc.local ~/dotfiles/
```

## karabiner

After install by `osx/homebrew/common.sh`.
Run Karabiner and accept permissions and do below.

```bash
osx/karabiner.sh
```
